Welcome to Fire JS

Ready To Use

Ready to use out of the box

Just `npm start' and it's ready to go

Any configuration you want is in `conf.json'

And the server rebuilds on updates

Simple

Escapable files in `/template/'

Static files in `/static/'

Helper functions and actions in `/module/'

Routes in `/routes'

Required packages and your configuration for them in `/require.js'

Add your own compiler for sass, typescript, etc in the `./script/startup' file so they rebuild on updates, don't forget to add, or change, the directory in nodemon

Or modify the server through `index.js' and the `/server/' directory

PHP but better

Like PHP you can escape to server side in any template file

But instead of PHP you use NodeJS

You can write to the file from server side with `res.echo(data);'

From server side you have access to other templates and modules

You also have access to the ExpressJS request and response, have a look at `package.json' to see the included middleware

Include templates by running them, `template(req, res);'

And use modules as you define them

The Generated Files

The `/src/' directory is where your templates compile to

`/router.js' is where your routes compile to actual code

`/system.js' is the amalgamation of `require.js', `/module/', and `/src'; this allows you not to need the NodeJS require api when accessing you own code
